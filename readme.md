## Introduction
@za-cli/react-ts是react组件构建工具，使用ts写法，开发入口文件为test/index.tsx，打包入口文件为src/index.tsx，生成文件为dist/js/index.js

## Installation
```
yarn global add @za-cli/react-ts
```

## Usage
```
// 生成react组件文件夹结构，安装@za-build/react-ts脚手架
za-react-init

// 进入创建的文件夹
cd yourProjectName

// 开发组件
za-react-dev

// 打包生产组件
za-react-build

// 上传组件
za-react-publish
```
